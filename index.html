🎨 MoruToon

Lightweight gimmick & particle shader collection for VRChat.

📥 Installation

Add this URL to VCC (VRChat Creator Companion):

https://moruton1119.github.io/MoruToon/index.json

Steps:

  1. Open VCC
  2. Go to Settings → Packages
  3. Click "Add Repository"
  4. Paste the URL above
  5. Click "Add" and accept the prompt
  6. Install MoruToon from your project's package list
