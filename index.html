📌 Manual VCC Setup

  1. Open VCC (VRChat Creator Companion)
  2. Go to Settings → Packages
  3. Click "Add Repository"
  4. Paste this URL:
  5. https://moruton1119.github.io/MoruToon/index.json
  6. Click "Add" and accept the prompt
  7. Install MoruToon from your project's package list
