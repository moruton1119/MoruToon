# MoruToon

Lightweight gimmick & particle shader collection for VRChat.

## Features

- **UV Scroll** - Animated texture scrolling for magic circles, energy flows
- **Emission** - HDR emission with pulse animation support
- **Dissolve** - Noise-based dissolve with glowing edge
- **Layer Blend** - Multi-texture blending (additive/multiply/screen)
- **Flipbook** - Texture sheet animation for explosions, fire, smoke
- **Soft Particle** - Distance-based alpha fade to prevent clipping
- **Stencil** - Portal and masking effects
- **Template System** - Tab-based preset switching in the Inspector

## Installation

### VCC (Recommended)

1. Open VCC (VRChat Creator Companion)
2. Go to Settings → Packages
3. Click "Add Repository"
4. Paste: `https://moruton1119.github.io/MoruToon/index.json`
5. Add the repository
6. Install "MoruToon" from your project's package list

### Manual

Download the latest `.unitypackage` from [Releases](../../releases) and import into your project.
